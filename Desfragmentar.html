Desfragmentar disco

Desfragmentar el disco, es un proceso que reorganiza los datos que se encuentran almacenados en un disco duro. Si bien la desfragmentacion se hace para un disco duro, a la unidad solida se le realiza una optimizacion ya que no es necesario una desfragmentacion en ella.

Como Desfragmentar y/o optimizar

Pasos para desfragmentar:

  • Abrir Ejecutar con las teclas Windows + R

    Luego de ello escribir dfrgui, como se muestra en la imagen y ACEPTAR.

  • Se les abrira la interfaz donde se podra desfragmentar

    Escoger todas las unidades mostradas para que lo desfragmente y optimice en caso de ser una unidad solida.
